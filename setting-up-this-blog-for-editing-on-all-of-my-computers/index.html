Setting up this blog for editing on all of my computers.

July 30, 2020

So yeah I'm using Publii for this and I have been really wringing my hands over it, but today I got editing of the content from multiple computers working well. Publii is like a really stripped down and simplified Wordpress, but instead of requiring an entire stack of server based tech, it is just a program you can install on Windows.

This might seem like a step backwards if you can only edit your stuff from one computer, but fortunately, it's possible to store the source data in Dropbox and sync it between more than one machine. 

Anyway I don't really want to make this entire blog about having a blog, but after facing nightmare scenario of using Jekyll and writing posts in Markdown (yeesh), this is amazing.
